# CampaignHub Client

React + Vite frontend for the CampaignHub landing page.

## Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Run

```bash
cd client
npm install
npm run dev
```

The dev server proxies `/api` requests to `http://localhost:5001`, so you can run the frontend beside the existing Express backend.

## Build

```bash
npm run build
```
