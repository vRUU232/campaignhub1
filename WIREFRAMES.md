## Color Scheme

| Element         | Color Code | Usage                              |
|-----------------|------------|------------------------------------|
| Primary         | `#4F46E5`  | Buttons, links, active states      |
| Primary Dark    | `#4338CA`  | Hover states                       |
| Success         | `#10B981`  | Sent, delivered, active status     |
| Warning         | `#F59E0B`  | Scheduled, pending status          |
| Danger          | `#EF4444`  | Failed, opted-out, delete actions  |
| Neutral Gray    | `#6B7280`  | Draft, secondary text              |
| Light Gray      | `#F3F4F6`  | Background, borders                |
| Dark            | `#111827`  | Primary text                       |
| White           | `#FFFFFF`  | Cards, backgrounds                 |

## Status Indicators

| Status      | Icon | Color   | Description                    |
|-------------|------|---------|--------------------------------|
| Sent        | 🟢   | Green   | Campaign sent successfully     |
| Delivered   | ✓✓   | Green   | Message delivered to recipient |
| Scheduled   | 🟡   | Yellow  | Campaign scheduled for later   |
| Draft       | ⚪   | Gray    | Campaign not yet sent          |
| Failed      | 🔴   | Red     | Message delivery failed        |
| Opted Out   | 🔴   | Red     | Contact opted out              |
| Active      | 🟢   | Green   | Contact can receive messages   |
| Inactive    | ⚪   | Gray    | Contact marked inactive        |

## Responsive Breakpoints

| Breakpoint | Width    | Layout Changes                          |
|------------|----------|----------------------------------------|
| Mobile     | < 640px  | Sidebar hidden, full-width content     |
| Tablet     | 640-1024px | Collapsible sidebar, 2-column grids  |
| Desktop    | > 1024px | Full sidebar, 4-column stats           |

## Component Library Recommendation

For building this UI, consider using:
- **Tailwind CSS** - For utility-first styling
- **Headless UI** - For accessible components
- **React Router** - For navigation
- **React Query** - For data fetching
- **Chart.js** or **Recharts** - For analytics charts
- **React Hook Form** - For form handling
